## 故障排除流程

当遇到系统问题时，请按以下流程进行排查：

### 1. 权限相关问题
- 401/403权限错误
- 中间件配置问题
- 角色权限验证失败

参考: [权限中间件修复报告](PERMISSION_MIDDLEWARE_FIX.md)

### 2. 缓存相关问题
- 页面路由不工作
- HMR不工作
- 组件更新问题

参考: [运维操作指南](../operations/CACHE_CLEANUP_GUIDE.md)

### 3. 调试和诊断
- 使用Chrome DevTools进行诊断
- 查看系统日志和错误信息

参考: [调试文档](../debug/)

## 报告新问题

如果在开发过程中遇到新的问题：

1. 记录详细的错误信息和复现步骤
2. 分析问题的根本原因
3. 提供解决方案和修复步骤
4. 将文档添加到本目录或相应的子目录

## 维护指南

- 确保所有故障排除文档包含完整的问题描述、解决方案和验证步骤
- 定期更新文档以反映最新的系统状态
- 保持文档的实用性和可操作性
