# UX设计规范详细说明

## 智能会议室管理系统 - UX设计核心原则

### 设计系统选择

#### PrimeVue + Aura主题 + 企业定制
- **选择理由**: 企业级组件库，专业商务外观，完整组件覆盖
- **核心组件**: DataTable, Calendar, FormKit, 状态指示器, Chart组件
- **定制需求**: 企业品牌色彩定制，飞书/Confluence风格交互模式

#### 色彩体系
```css
/* 专业商务蓝主题 */
--primary-color: #1e40af;      /* 主色调 - 主要按钮、重要操作、导航 */
--secondary-color: #3b82f6;    /* 辅助色 - 次要按钮、链接、强调元素 */
--success-color: #059669;      /* 成功色 - 可用状态、成功操作、确认 */
--warning-color: #d97706;      /* 警告色 - 警告提示、注意事项 */
--error-color: #dc2626;        /* 错误色 - 错误状态、冲突提示 */
--neutral-color: #6b7280;      /* 中性色 - 文本、边框、辅助信息 */

/* 语义化颜色应用 */
.room-available { background: var(--success-color); color: white; }
.room-occupied { background: var(--error-color); color: white; }
.room-maintenance { background: var(--warning-color); color: var(--neutral-color); }
.room-reserved { background: var(--secondary-color); color: white; }
```

### 核心用户体验原则

#### 1. 速度原则 - 关键操作<3秒完成
- 预约流程必须极速
- 状态查询即时响应  
- 页面切换流畅无缝

#### 2. 指导原则 - 智能引导，减少思考
- 智能推荐最优选项
- 步骤清晰明确
- 错误预防优于错误处理

#### 3. 灵活原则 - 简单操作，强大功能
- 基础操作极简
- 高级功能可选
- 视图模式自由切换

#### 4. 反馈原则 - 实时响应，状态清晰
- 操作立即反馈
- 状态实时更新
- 结果清晰可见

### 定义性体验: "预约会议室，从未如此简单"

#### 3步完成预约的极致流程
1. **智能发现 (5秒内完成)**
   - 时间优先输入
   - 会议室可用性一目了然
   - 智能推荐算法
   - 简化筛选器

2. **快速选择 (3秒内完成)**
   - 一击预约
   - 实时冲突检测
   - 相似推荐
   - 预览模式

3. **智能确认 (10秒内完成)**
   - 智能表单预填
   - 一键操作
   - 即时通知
   - 添加到日历

### 组件库架构

#### 标准PrimeVue组件
- **DataTable**: 会议室列表管理 (支持排序、筛选、分页)
- **Calendar**: 预约时间选择 (月视图/周视图/日视图)  
- **Button**: 各类操作按钮 (主要/次要/文本按钮)
- **InputText**: 文本输入 (搜索、表单填写)
- **Dropdown**: 选择器 (会议室选择、时间选择)
- **Dialog**: 弹窗 (预约确认、详情查看)
- **Toast**: 消息提示 (成功/错误/警告)
- **Card**: 卡片容器 (会议室信息展示)
- **Badge**: 状态标识 (可用/占用/维护)

#### 定制企业组件

##### 1. SmartRoomCard (智能会议室卡片)
```vue
<SmartRoomCard
  :room="roomData"
  :recommended="true"
  :showQuickActions="true"
  @reserve="handleReserve"
  @viewDetails="handleViewDetails"
/>
```
**功能特性**:
- 展示会议室核心信息 + 智能推荐标识
- 支持一键预约、状态实时更新、历史使用标识
- 响应式设计，移动端触控优化

##### 2. TimeRangePicker (智能时间选择器)
```vue
<TimeRangePicker
  v-model="selectedTime"
  :availableSlots="timeSlots"
  :conflictDetection="true"
  @change="handleTimeChange"
/>
```
**功能特性**:
- 可视化时间选择 + 冲突检测
- 智能推荐时段、冲突自动解决
- 拖拽交互支持

##### 3. ReservationWizard (预约流程向导)
```vue
<ReservationWizard
  :steps="['selectRoom', 'selectTime', 'confirm']"
  :smartRecommendations="true"
  @complete="handleReservationComplete"
/>
```
**功能特性**:
- 3步预约流程引导
- 步骤回退、智能预填、进度指示
- 表单验证和错误处理

##### 4. RoomStatusIndicator (会议室状态指示器)
```vue
<RoomStatusIndicator
  :status="roomStatus"
  :occupancy="occupancyRate"
  :nextAvailable="nextAvailableTime"
  :realTime="true"
/>
```
**功能特性**:
- 实时状态显示
- 颜色语义化、动画效果、多设备同步
- 状态变更通知

### UX模式决策

#### 按钮层次系统
- **主要操作**: 蓝色背景 (#1e40af)，14px字体，用于"立即预约"等核心操作
- **次要操作**: 灰色背景 (#f3f4f6)，用于"查看详情"等辅助操作  
- **文本操作**: 蓝色文字 (#3b82f6)，用于链接式操作
- **危险操作**: 红色背景 (#dc2626)，用于"取消预约"等破坏性操作

#### 反馈模式
- **成功反馈**: 绿色Toast + 勾选图标，自动3秒消失
- **错误反馈**: 红色Toast + 错误图标，需要用户确认
- **警告反馈**: 橙色Toast + 警告图标，5秒自动消失
- **信息反馈**: 蓝色Toast + 信息图标，3秒自动消失
- **加载状态**: 骨架屏 + 进度条，提供操作反馈

#### 表单模式
- **标签位置**: 输入框上方，左对齐
- **必填标识**: 红色星号(*)标识
- **验证时机**: 失焦时验证，实时反馈错误
- **错误显示**: 输入框下方红色文字提示
- **帮助文本**: 问号图标tooltip，悬停显示详细说明

#### 确认模式
- **删除操作**: 始终需要二次确认
- **未保存提醒**: 页面离开时自动保存提醒
- **不可逆操作**: 详细说明后果的确认框
- **批量操作**: 显示影响数量的确认

#### 搜索模式
- **触发**: 输入时自动搜索，500ms防抖
- **结果**: 即时显示匹配结果
- **筛选**: 侧边栏筛选器，支持多选
- **无结果**: 搜索建议 + 搜索历史

### 响应式设计策略

#### 断点策略
- **Mobile**: < 768px (手机竖屏)
- **Tablet:** 768px - 1024px (平板、手机横屏)  
- **Desktop:** > 1024px (桌面、大屏)

#### 适配模式

**桌面端 (>1024px)**
- **布局**: 混合模式布局
- **导航**: 顶部导航 + 侧边栏
- **内容**: 2-3列网格布局
- **交互**: 鼠标悬停效果，键盘导航支持

**平板端 (768-1024px)**
- **布局**: 简化的混合模式
- **导航**: 顶部导航 + 汉堡菜单
- **内容**: 1-2列布局，卡片式设计
- **交互**: 触控优化，保留悬停效果

**移动端 (<768px)**
- **布局**: 移动优先模式
- **导航**: 底部标签导航
- **内容**: 单列布局，大按钮设计
- **交互**: 纯触控操作，手势支持

#### 移动端专用优化
- **触控区域**: 最小44×44px触控区域
- **手势支持**: 左滑预约、右滑查看详情
- **底部导航**: 会议室、日历、统计、我的
- **大按钮设计**: 45px最小触控区域

### 可访问性策略

#### WCAG 2.1 AA 级合规
- **颜色对比**: 文字与背景对比度≥4.5:1
- **触控目标**: 最小44×44px触控区域
- **键盘导航**: 全键盘可操作，Tab键逻辑清晰
- **焦点指示**: 明显的焦点状态，2px蓝色边框

#### 屏幕阅读器支持
- **语义化HTML**: 正确使用标题、列表、按钮等标签
- **ARIA标签**: 为动态内容添加aria-label和aria-live
- **跳转链接**: 页面开头添加"跳转到主内容"链接
- **图片替代**: 所有有意义的图片提供alt文本

#### 其他可访问性特性
- **字体缩放**: 支持200%字体缩放不影响布局
- **高对比度**: 支持系统高对比度模式
- **动画控制**: 提供"减少动画"选项
- **错误处理**: 清晰的错误信息和恢复指导

### 关键用户旅程设计

#### 快速预约流程 (3步完成)

**步骤1: 智能发现**
- 时间优先输入设计
- 会议室状态可视化 (✅可用 ❌占用 ⏳即将可用)
- 基于历史数据的智能推荐算法
- 按需显示的筛选器 (容纳人数、设备需求、位置偏好)

**步骤2: 快速选择**  
- 一击预约的推荐会议室
- 实时冲突检测和替代方案显示
- 基于历史的相似会议室推荐
- 鼠标悬停显示完整信息的预览模式

**步骤3: 智能确认**
- 自动预填常用信息的智能表单
- 支持常用场景的一键预约
- 预约成功的即时通知和自动日历添加
- 会议材料和参会人员的快速配置
