# 任务完成检查清单

## 代码开发完成后

### 1. 代码质量检查
```bash
# 运行ESLint检查
npm run lint

# 运行类型检查
npm run type-check

# 格式化代码
npm run format
```

### 2. 测试相关
```bash
# 运行单元测试
npm run test

# 检查测试覆盖率
npm run test:coverage
```

### 3. 功能验证
- 手动测试新功能
- 检查UI组件响应式布局
- 验证表单验证逻辑
- 测试错误处理

### 4. 数据库相关 (如涉及)
```bash
# 检查Prisma schema
npm run db:generate

# 如有数据模型变更，运行迁移
npm run db:migrate
```
